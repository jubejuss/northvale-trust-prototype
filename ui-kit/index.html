Three radii: small for marks, medium for controls, large for cards and panels.

Buttons

Three variants: .btn--primary, .btn--secondary, .btn--ghost. Add the base .btn class to any <a> or <button>.

Label + control pattern using .form-field and .form-field__control. Works for inputs and textareas.
